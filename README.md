# Personalized-Workout-Diet-Planner-with-AI-
AI Fitness &amp; Diet Planner: A Streamlit app using LangChain &amp; Groq Cloud API (Llama 3.3) to generate personalized workout and meal plans based on your profile. Features an interactive chat to ask questions about your plan. Requires a free API key.

🚀 How to Run the Application Follow these steps to get the app fully running on your local machine:

1 .Create a project folder and place the app.py and requirements.txt files inside it.

Install Python dependencies by opening a terminal in your project folder and running: s

pip install -r requirements.txt Get a Groq API Key (it's free):

Visit the Groq Cloud Console.

Sign up or log in.

Navigate to the "API Keys" section and create a new key.

reate a .env file in the same project folder and add your key:

GROQ_API_KEY=your_actual_groq_api_key_here

Run the Streamlit app from the terminal: streamlit run app.py Use the application in your browser. Enter your details (age, weight, diet type, restrictions, etc.) and click "Generate Plans".
